# Aretex PH Repository
This repository contains all of Aretex PH. Visit it at https://aretex-ph.com/

# Development
## Initial Setup
Install the following:
- **NodeJS** - https://nodejs.org/en
- **Docker** - https://www.docker.com/

First, clone the repository to a folder. Then, run the following:

```bash
npm i
```

## Run development
Run the command to start a development server:
```bash
npm run dev
```

Then, open the server at [http://localhost:3000](http://localhost:3000).
